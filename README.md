# VR-Physics-Integration
## Description 
A Virtual Reality scenario where a particle physics engine is used to emulate soft/deformable bodies, liquids, clothes and ropes.

### Topics:
- VR Virtual-Reality 
- Physics Simulation

## Requirements & Dependencies
- Unity 2021.3
- HTC Vive

## Install / Run
*   clone the repository
*   open with UnityHub
*   connect your device
*   run the app
